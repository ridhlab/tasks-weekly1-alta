Section 6 - Bootstrap

Pengertian

Bootstrap adalah salah satu framework CSS yang populer. Bootstrap menyediakan beberapa class sebagai utility sehinggan mendukung pembuatan website dengan cepat. Bootstrap banyak digunakan terutama ketika baru terjun di dunia pengembangan website. Banyak orang merujuk bootstrap sebagai pilihan yang tepat ketika mempelajari framework CSS. Alasannya tidak lain karena simpel dan mudah dipelajari, serta mendukung beberapa komponen yang sering digunakan. Komponen seperti button, navbar, dan card dapat kita temukan disini

Bootstrap mendukung halaman yang responsive, artinya ia bisa menyesuaikan halamannya akan tampil seperti apa jika dibuka di device yang berbeda. Bootstrap juga mendukung function-function javascript yang memudahkan kita untuk membuat tampilan website yang menarik dan interaktif.

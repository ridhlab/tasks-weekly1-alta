Section 7 - JavaScript Refreshment

Pengertian JavaScript

JavaScript adalah bahasa pemrograman high level yang digunakan untuk mengembangkan web. High level programming berarti bahasanya lebih mudah untuk dipahami manusia karena dekat dengan bahasa manusia. JavaScript kompatibel dengan semua modern browser yang ada sekarang sehingga hal inilah yang menyebabkan kenapa banyak orang menggunakan JavaScript.

JavaScript mengeluarkan ECMAScript versi 6, atau biasa disingkat ES6 di tahun 2015 sebagai pembaharuan dari JavaScript versi lama. Terdapat beberapa penambahan fitur di ES6 dengan tujuan supaya ES6 bisa menangani pembuatan website yang lebih kompleks.

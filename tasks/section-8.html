Section 8 - Clean Code

Pengertian Clean Code

Dalam proses pengembangan website, tentu kita akan berkolaborasi dalam sebuah tim yang terdiri dari beberapa orang. Untuk memudahkan proses pengembangan, tentu harus ada aturan ketika menulis code program. Seperti memberi nama variabel dengan jelas sesuai konteksnya, dan tidak mengulang-ulang kode. Konsep pengerjaan seperti itu adalah implementasi dari clean code.

Clean code adalah istilah untuk membuat kode yang mudah dibaca, dipahami, dan dimodifikasi oleh programmer. Karena programmer yang baik adalah programmer yang mampu membuat kode yang tidak hanya dipahami oleh komputer, tetapi juga manusia.

Clean code membuat proses pengembangan lebih cepat dan lebih kondusif. Karena dengan clean code, kita bisa meminimalisir kemungkinan buruk yang akan terjadi.
